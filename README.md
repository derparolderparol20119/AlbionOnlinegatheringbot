lbion Online Resource Bot — это бот для автоматизации сбора ресурсов в Albion Online.

Бот отслеживает ресурсы на экране с помощью положения курсора мыши и выполняет действия по их сбору. 
При необходимости он также способен атаковать и уничтожать мобов, мешающих добыче ресурсов.

✨ Возможности
   Автоматический сбор ресурсов.
   Отслеживание ресурсов с помощью мыши.
   Автоматическая атака и убийство мобов.
   Непрерывный цикл поиска и добычи ресурсов.
